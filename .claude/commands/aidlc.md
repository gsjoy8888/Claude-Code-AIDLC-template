# AI-DLC Workflow

Start the AI-DLC (AI-Driven Development Life Cycle) workflow to guide structured software development.

Usage: /aidlc [your project description]

Example:
- /aidlc Create a task management REST API with user authentication
- /aidlc Add real-time notification to my existing chat app
- /aidlc Investigate and fix performance issues in the user dashboard

This command activates the AI-DLC workflow defined in CLAUDE.md, starting from workspace detection and guiding you through all phases: Inception → Construction → Operations.
