# ASCII 图表标准

## 强制要求：仅使用基本 ASCII 字符

**关键要求**：始终使用基本 ASCII 字符绘制图表（最大兼容性）。

### ✅ 允许使用：`+` `-` `|` `^` `v` `<` `>` 和字母数字文本

### ❌ 禁止使用：Unicode 制表符
- 禁止：`┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 原因：在不同字体/平台上渲染不一致

## 标准 ASCII 图表模式

### 关键要求：字符宽度规则
**方框中的每一行必须具有完全相同的字符数（包括空格）**

✅ 正确（所有行 = 67 个字符）：
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 错误（宽度不一致）：
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

## 验证

创建图表前：
- [ ] 仅使用基本 ASCII：`+` `-` `|` `^` `v` `<` `>`
- [ ] 不使用 Unicode 制表符
- [ ] 使用空格（不使用制表符）对齐
- [ ] 角落使用 `+`
- [ ] **所有方框行具有相同的字符宽度**（计算包括空格在内的字符数）
- [ ] 测试：在等宽字体中验证角落垂直对齐
